MK K School PWA Installation

Make your website installable on desktop and mobile devices

Instructions

1

Create manifest.json

Create a file named manifest.json in your website's root directory with the provided code.

2

Link the manifest

Add the following code to the <head> section of your HTML pages:

<link rel="manifest" href="/manifest.json">
3

Add icons

Make sure to add the icon files in the specified sizes to your server in the paths mentioned in the manifest.

4

Test your PWA

Use your browser's developer tools to test if your Progressive Web App is configured correctly.

Note: For security reasons, your website must be served over HTTPS to use PWA features like service workers.
