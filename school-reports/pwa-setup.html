Fix missing install icon on your dashboard

Why isn't the install icon showing?

Common reasons the install prompt might not appear:

  • Manifest.json is not properly linked
  • Icons are missing or paths are incorrect
  • Service worker is not registered
  • HTTPS requirement not met (for production)
  • PWA criteria not fully met

Fixing the Issue

1

Verify Manifest Link

Check that the manifest is properly linked in your HTML:

<link rel="manifest" href="/manifest.json">
2

Check Icon Paths

Ensure all icon paths in your manifest.json are correct and the files exist.

3

Register a Service Worker

Add this code to your page to register a service worker (required for PWA):

if ('serviceWorker' in navigator) {
  window.addEventListener('load', function() {
    navigator.serviceWorker.register('/sw.js')
      .then(function(registration) {
        console.log('SW registered: ', registration);
      })
      .catch(function(registrationError) {
        console.log('SW registration failed: ', registrationError);
      });
  });
}
4

Trigger Install Prompt Manually

You can add a button to trigger the install prompt manually:

let deferredPrompt;
const installButton = document.getElementById('installButton');

window.addEventListener('beforeinstallprompt', (e) => {
  e.preventDefault();
  deferredPrompt = e;
  installButton.style.display = 'block';
});

installButton.addEventListener('click', async () => {
  if (deferredPrompt) {
    deferredPrompt.prompt();
    const { outcome } = await deferredPrompt.userChoice;
    if (outcome === 'accepted') {
      deferredPrompt = null;
    }
  }
});

Note: For security reasons, your website must be served over HTTPS to use PWA features like service workers in production.

Basic Service Worker (sw.js)

Create a file named sw.js with this content:

const CACHE_NAME = 'mkk-school-v1';
const urlsToCache = [
  '/',
  '/school-reports/auth/login.php',
  '/school-reports/super_admin/dashboard.php',
  '/styles.css',
  '/app.js'
];

self.addEventListener('install', function(event) {
  event.waitUntil(
    caches.open(CACHE_NAME)
      .then(function(cache) {
        return cache.addAll(urlsToCache);
      })
  );
});

self.addEventListener('fetch', function(event) {
  event.respondWith(
    caches.match(event.request)
      .then(function(response) {
        if (response) {
          return response;
        }
        return fetch(event.request);
      }
    )
  );
});
